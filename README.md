# "BillieJeansAuto-API"

### Description

This repository serves as the back-end for "BillieJeansAuto" (a basic employee/task ticketing system).

---

- [Express.js](https://expressjs.com)
- [React.js](https://reactjs.org)
- [Node.js](https://nodejs.org/)
